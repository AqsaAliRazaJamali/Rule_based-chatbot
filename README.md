# 🤖 ChatSimple
A rule-based Python chatbot with a Tkinter GUI.

### 🌟 Key Features:
* **Math Solver:** Uses MathJS API for calculations.
* **Knowledge Base:** Info on OS, Sukkur IBA, and Dadu.
* **Fun Mode:** Real-time jokes and facts via API.
* **Motivation:** Study quotes and inspirational stories.

### 🛠️ Setup
1. Install dependencies: `pip install requests wikipedia`
2. Run the script: `python chatbot.py`
